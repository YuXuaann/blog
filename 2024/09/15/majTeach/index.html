麻将小课堂~

2024/9/15 teach majsoul 麻将 雀魂

快来玩雀魂喵~

1.麻将牌的种类

数牌

万子牌

从左到右为一万~九万

饼子牌

从左到右为一饼~九饼, 也叫筒子，可能对初学者来说七饼有点难以辨别

索子牌

从左到右为一索~九索，也叫条子，对初学者来说别把二索和一索搞混了，八索比较特殊

字牌

风牌

从左到右为东南西北

三元牌

从左到右为白发中


注意事项

  • 在日麻中没有花牌（梅兰竹菊、春夏秋冬）
  • 请记住上述展示牌的顺序，尤其是三元牌，不同于国内熟悉的顺序（中发白），在日麻中是白发中，这个顺序在后文的宝牌中会有用到

基本牌型

顺子


由同种三张连续的数牌组成，如2-3-4，4-5-6

刻子


由同种三张相同的任意牌组成，如3-3-3，南-南-南（南上加南啊）

雀头


由两张相同的任意牌组成，如5-5，东-东

注意事项

  • 组成顺子的三张牌必须是同种牌，万子、饼子、索子不能混合组成顺子
  • 组成顺子的三张牌不能跨越边界，如8-9-1是不合法的顺子

2.日麻的基本规则

概述：四个人一桌，每人13张牌，轮流摸牌打牌（还有吃、碰、杠、胡这些操作），组建成能够和牌的牌型，最终和牌获得分数。比赛有多局，在所有局结束后，依据分数结算排名
以上的专业术语后文均会有解释滴~

游戏术语

请逐条阅读，有递进关系哦~

  • 面子：在牌型中，顺子、刻子统称为面子
  • 役：在日麻中，满足一定条件的牌型称为役，新手需要记住的常见役种有：立直、断幺九、自风牌、场风牌、白发中 ( 后文有详细介绍哦~ )
  • 和牌条件：当你的牌中有四个面子和一个雀头并且有役时，才可以和牌，特殊役种除外 ( 后文有补充滴~ )

  • 上家：在游戏中，你的左边的玩家称为上家
  • 下家：在游戏中，你的右边的玩家称为下家
  • 吃：当你的牌中有一个顺子缺其中一张时，而上家恰好打出那张牌，你可以吃上家打出的牌，组成一个顺子
  • 碰：当你的牌中有两张相同的牌，其他家打出了一张相同的牌时，你可以碰他打出的牌，组成一个刻子
  • 杠：当你的牌中有三张相同的牌，其他家打出了一张相同的牌时，你可以杠他打出的牌，组成一个杠子
  • 副露：吃、碰、杠的统称

  • 听牌：当你的牌差一张牌就可以和牌时，称为听牌
  • 荣和：当别人打出的牌使你的牌满足和牌条件，你可以和牌，获得本局胜利，国内又叫胡牌
  • 放铳(点炮)：打出的牌使某家荣和。例如，A打出一张牌，B荣和，那么A就 放铳(点炮) 了，或称为A 放铳(点炮) B
  • 和牌：荣和、放铳(点炮) 的统称
  • 自摸：当你自己摸到的牌使你的牌满足和牌条件，你可以自摸，获得本局胜利

  • 振听：简而言之就是无法荣和的状态

    • 当你的牌中有四个面子和一个雀头但是无役时，称为无役振听，不仅不能荣和，还不能自摸，新手特别需要注意这一点哦，和牌需有役！

    这是新手最爱犯的错误拉！
    • 当你听的牌为你打出过的牌时，称为舍牌振听，不能荣和，只能自摸
    • 当你听牌时，别人打出一张牌使你满足了和牌条件，但是你没有选择荣和，那么直到下一次你摸牌前，你都是同巡振听状态，不能荣和
    • 特别的，当你处于立直状态时，满足了和牌条件但你不和，那么进入立直振听状态，也就是永久振听，不能荣和，只能自摸

  • 庄：庄家。庄家的分数变动相较于子家是1.5倍的，庄家和牌或流局听牌时，继续连庄，否则下一局换下家做庄
  • 场风：每一圈(四个人都做过一次庄)所代表的风牌，第一圈为东风，第二圈为南风…
  • 自风：每个玩家的风牌，东南西北，东风为庄家的门风，然后按照逆时针顺序依次为东南西北
  • 流局：牌摸完了也没人和牌，称为流局，流局时，若庄家听牌，庄不变继续下一局

游戏的大致流程

每人起始有25000分，进行若干局比赛，以东风局为例，每个人做过一次庄后，或当有人分数小于0时，游戏结束，结算分数，依据分数结算排名

计算分数

概述：看看你到底有几番，役种的番数加上宝牌的番数

  • 简单役种介绍

    • 立直：在你未副露听牌的情况下，你可以打出一张牌、支付一千点并宣布立直，那么你进入立直状态，接下来所有你的摸牌要么自摸，要么直接打出，一番

    • 断幺九：手牌中没有一、九、字牌，一番
    • 自风牌：有自风牌成为了刻子，一番
    • 场风牌：有场风牌成为了刻子，一番
    • 白发中：有白发中成为了刻子，每有一个刻子一番

  • 宝牌：每一个加一番

    • 表宝牌：游戏左上角显示的牌的下一张牌（如果是数牌9，那宝牌就是同种数牌1），如果是字牌，按照白发中、东南西北这个轮回顺序
    • 里宝牌：立直和牌后才会看到的牌，同表宝牌的规则
    • 红宝牌：红色的5万，5饼，5索

其它事项

  • 上述均为四人麻将规则，也有三人麻将规则，但是基本规则是一样的
  • 无役状态下，即使有宝牌也不能和牌，千万要记得这点哦~
